কিভাবে SSH Key Create করব এবং তা Setup করব

SSH => Secure Shell

SSH => এটি হচ্ছে একটি Communication Protocol যার মাধ্যমে একটি কম্পিউটার অন্য একটি কম্পিউটারের সাথে Securely Connection তৈরি করতে পারে।

SSH দিয়ে Clone করতে হলে প্রথমে Terminate এর Git Clone Url দিতে হবে।

Public SSH Key Check করার জন্য এর Terminate Root Directory তে যেতে হবে। তারপর Ls -a দিলে সকল Hidden Folder সহ দেখা যাবে । তারপর cd .ssh দিতে হবে আপনার কম্পিউটারে সে ফাইলটি আছে কিনা চেক করার জন্য। যদি থাকে তাহলে সে ফাইলে SSH Key Setup করবে।

SSH Public Key Generate

ssh -> ssh-keygen -o -t rsa -C github email

তারপর ssh key একটি specific location এ set করতে হবে।
